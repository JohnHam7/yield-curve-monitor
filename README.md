## Overview

This project monitors the U.S. Treasury yield curve to detect inversion signals and analyze their relationship with macroeconomic conditions and recession risk.

The 2s10s spread (10Y minus 2Y Treasury yield) is one of the most widely tracked recession leading indicators. This dashboard tracks its historical behavior, flags inversion periods, and quantifies anomalies using a rolling z-score.

---

## Key Features

- **2s10s Spread Tracking** вҖ” Real-time spread calculation with inversion detection
- **NBER Recession Overlay** вҖ” Visual comparison of inversion periods vs. official recessions
- **Yield Curve Shape Analysis** вҖ” Full maturity spectrum (3MвҖ“30Y) across different rate environments
- **Z-Score Anomaly Detection** вҖ” Statistical identification of historically extreme spread levels

pythonimport pandas as pd
import time


def get_yield_data(fred, start_date='2000-01-01'):
    # л§Ңкё°лі„ Treasury yield мҲҳм§‘
    maturities = {
        '3M': 'GS3M', '1Y': 'GS1', '2Y': 'GS2',
        '5Y': 'GS5', '7Y': 'GS7', '10Y': 'GS10',
        '20Y': 'GS20', '30Y': 'GS30'
    }
    series_dict = {}
    for label, code in maturities.items():
        series_dict[label] = fred.get_series(code, observation_start=start_date)
        time.sleep(0.3)

    df = pd.DataFrame(series_dict).dropna()
    return df


def calculate_spread(curve_df):
    # 2s10s spread л°Ҹ inversion м—¬л¶Җ кі„мӮ°
    df = curve_df.copy()
    df['spread'] = df['10Y'] - df['2Y']
    df['inverted'] = df['spread'] < 0
    return df


def calculate_zscore(series, window=24):
    # Rolling z-score кі„мӮ°
    mean = series.rolling(window).mean()
    std = series.rolling(window).std()
    return (series - mean) / std


def get_recession_data(fred, start_date='2000-01-01'):
    # NBER recession лҚ°мқҙн„° мҲҳм§‘
    return fred.get_series('USREC', observation_start=start_date).fillna(0)

## Overview

This project monitors the U.S. Treasury yield curve to detect inversion signals and analyze their relationship with macroeconomic conditions and recession risk.

The 2s10s spread (10Y minus 2Y Treasury yield) is one of the most widely tracked recession leading indicators. This dashboard tracks its historical behavior, flags inversion periods, and quantifies anomalies using a rolling z-score.

---

## Key Features

- **2s10s Spread Tracking** вҖ” Real-time spread calculation with inversion detection
- **NBER Recession Overlay** вҖ” Visual comparison of inversion periods vs. official recessions
- **Yield Curve Shape Analysis** вҖ” Full maturity spectrum (3MвҖ“30Y) across different rate environments
- **Z-Score Anomaly Detection** вҖ” Statistical identification of historically extreme spread levels

## Key Findings

- The 2s10s spread inverted in **2000, 2006вҖ“07, and 2022вҖ“23**, each preceding or coinciding with economic stress
- The **2022вҖ“23 inversion was the deepest on record** (~-1.0%p), yet no NBER recession has been declared вҖ” a contested anomaly in current macro discourse
- The rolling z-score contextualizes each inversion within its recent rate environment, avoiding distortion from structurally different rate regimes
